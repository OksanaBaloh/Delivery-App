# Delivery-App
Description:
Web application where users can order food delivery.

Requirements:
For install and using server you have download npm.

After that __npm install__: express, body-parser, cors, sqlite3, md5.

How to launch:

For start server open folder "server" and use

npm start server

Finally, open "index.html" in any browser.

For close server use "CTRL+C"
